# Example Route A

Use this example when the user wants a Timberborn-style result and custom textures are not required.

## Scenario

- Source asset: a small decorative hut with clean geometry splits
- Goal: fast in-game placement with built-in Timberborn materials
- Tools: Blender CLI, `.timbermesh`, Blueprint, `TemplateCollection`

## Recommended Output

- one finished `.timbermesh`
- one building Blueprint
- one icon
- one `TemplateCollection` registration entry
- stock unfinished construction base

## Why Route A Fits

- geometry carries the look
- the user does not need the original texture
- the build can stay lightweight and avoid Unity
